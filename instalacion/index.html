Instalación

Ubuntu (13.10) pelo en pecho edition

{% highlight console %} $ sudo apt-get install build-essential git nodejs $ sudo apt-get install libsqlite3-dev libxslt1-dev libxml2-dev libssl-dev libreadline-dev libyaml-dev $ git clone https://github.com/sstephenson/rbenv.git ~/.rbenv $ git clone https://github.com/sstephenson/ruby-build.git ~/.rbenv/plugins/ruby-build $ git clone https://github.com/sstephenson/rbenv-gem-rehash.git ~/.rbenv/plugins/rbenv-gem-rehash $ echo 'export PATH="$HOME/.rbenv/bin:$PATH"' >> ~/.bashrc $ echo 'eval "$(rbenv init -)"' >> ~/.bashrc {% endhighlight %}

Cerrar el terminal y abrirlo nuevamente (para que se ejecute lo que agregamos en .bashrc)

{% highlight console %} $ rbenv install 2.1.1 $ rbenv global 2.1.1 $ echo "gem: --no-ri --no-rdoc" > ~/.gemrc $ gem update --system $ gem install rails $ gem install sqlite3 {% endhighlight %}

Ubuntu (13.10) puro Apt-Get

{% highlight console %} $ sudo apt-get install build-essential nodejs $ sudo apt-get install ruby2.0 ruby2.0-dev $ sudo apt-get install libsqlite3-dev libxslt1-dev libxml2-dev libyaml-dev $ sudo gem update --system $ sudo gem install rails $ sudo gem install sqlite3 {% endhighlight %}

Archlinux pelo en pecho edition

{% highlight console %} $ sudo pacman -S base-devel nodejs git $ sudo pacman -S libxslt libxml2 libyaml openssl readline zlib $ git clone https://github.com/sstephenson/rbenv.git ~/.rbenv $ git clone https://github.com/sstephenson/ruby-build.git ~/.rbenv/plugins/ruby-build $ git clone https://github.com/sstephenson/rbenv-gem-rehash.git ~/.rbenv/plugins/rbenv-gem-rehash $ echo 'export PATH="$HOME/.rbenv/bin:$PATH"' >> ~/.bashrc $ echo 'eval "$(rbenv init -)"' >> ~/.bashrc {% endhighlight %}

Cerrar el terminal y abrirlo nuevamente (para que se ejecute lo que agregamos en .bashrc)

{% highlight console %} $ curl -fsSL https://gist.github.com/mislav/a18b9d7f0dc5b9efc162.txt | rbenv install --patch 2.1.1 $ rbenv global 2.1.1 $ echo "gem: --no-ri --no-rdoc" > ~/.gemrc $ gem update --system $ gem install rails $ gem install sqlite3 {% endhighlight %}

Archlinux puro Pacman

{% highlight console %} $ sudo pacman -S base-devel nodejs $ sudo pacman -S ruby $ sudo pacman -S libxslt libxml2 libyaml $ sudo gem update --system $ sudo gem install rails $ sudo gem install sqlite3 {% endhighlight %}
